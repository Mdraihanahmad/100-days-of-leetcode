# Day 48 - Rotate Image

## Problem
Rotate an n x n matrix by 90 degrees clockwise in-place.

## Approach
1. Transpose matrix
2. Reverse each row

## Key Concept
Transpose + Reverse = Rotation

## Time Complexity
O(n²)

## Space Complexity
O(1)

## Example
Input:
[[1,2,3],
 [4,5,6],
 [7,8,9]]

Output:
[[7,4,1],
 [8,5,2],
 [9,6,3]]
