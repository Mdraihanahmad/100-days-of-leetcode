# Day 43 - Multiply Strings

## Problem
Multiply two large numbers represented as strings.

## Approach
- Simulate manual multiplication
- Store results in vector
- Handle carry separately
- Convert final vector to string

## Key Concept
String multiplication without BigInteger

## Time Complexity
O(n × m)

## Space Complexity
O(n + m)

## Example
Input:
"123", "456"

Output:
"56088"
