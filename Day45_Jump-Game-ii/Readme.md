# Day 45 - Jump Game II

## Problem
Find minimum jumps needed to reach last index.

## Approach
Use Greedy:
- Track farthest reachable index
- Increase jump count when current range ends

## Key Concept
Expand reachable range level by level.

## Time Complexity
O(n)

## Space Complexity
O(1)

## Example
Input:
[2,3,1,1,4]

Output:
2
